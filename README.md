Vimer
===========

  
## 安装

  $ git clone https://github.com/xiebiao/vimer  
  $ cd vimer  
  $ ./install.sh

## 通过Vundle安装插件

  使用 [vundle](https://github.com/gmarik/vundle) 管理插件 
  插件安装: PluginInstall
